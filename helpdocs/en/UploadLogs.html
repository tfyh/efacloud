Log Files

When efa is executed, log files are generated locally on the PC. which are transferred to the server as required as part of the upload operation. server as part of the upload operation. As a rule, the synchronisation errors and the efaCloud log file are transferred. To reduce the upload data volumes, especially with low bandwidths, only the synchronisation alternatively only the synchronisation error file or no log file can be uploaded. can be uploaded.

Settings

all = synchronisation errors and the efaCloud log file, errors = synchronisation errors, none = no file
