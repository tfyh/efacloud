# efacloud

[![Continuous Integration](https://github.com/tfyh/efacloud/actions/workflows/continuous-integration.yml/badge.svg)](https://github.com/tfyh/efacloud/actions/workflows/continuous-integration.yml)
[![Security Scanning](https://github.com/tfyh/efacloud/actions/workflows/security-scanning.yml/badge.svg)](https://github.com/tfyh/efacloud/actions/workflows/security-scanning.yml)

This repository contains the efacloud server PHP code.
The efacloud environment is a useful _extention_ to the [efa software](http://efa.nmichael.de/), written by [Nicholas Michael](http://www.nmichael.de/).

## Installation 

An installation overview can be found [here](https://www.efacloud.org/src/).
The latest release will also be published here in the [Release section](https://github.com/tfyh/efacloud/releases)

## Further material

Can be found [here](https://www.efacloud.org/material.html).

## Development

For developers who want to contribute to efaCloud or run tests locally, see the [Testing Documentation](tests/README.md).

**Note:** Development and testing files are automatically excluded from release downloads. If you download a ZIP from the releases page, you will get a clean installation package without test infrastructure. Developers who clone the repository will have access to the full test suite.
